Como usar este handbook

É um site 100% estático: você só edita este arquivo index.html, não precisa de servidor, Traefik nem Portainer.

  1. Edite os textos das seções abaixo com seus passos e prints.
  2. Para publicar grátis, suba este arquivo em um repositório do GitHub e ative GitHub Pages.
  3. (Opcional) Aponte um subdomínio (ex.: handbook.captioai.com) via CNAME para o seu Pages.

VPS padrão (Traefik + Portainer)

Use quando for montar o ambiente base de um cliente.

  1. Entrar por SSH como root (sudo -i).
  2. Atualizar pacotes: 
    apt-get update -y && apt-get upgrade -y && apt-get install -y curl ca-certificates
  3. Rodar instalador CAPTIO e escolher Traefik e Portainer no menu: 
    bash <(curl -fsSL https://raw.githubusercontent.com/cfjacques/captio-setup/v1.0/SetupCaptio)

Chatwoot (Instagram/Facebook)

  • No fluxo do Chatwoot o instalador pergunta (opcional): FB_APP_ID, FB_APP_SECRET, FB_VERIFY_TOKEN, IG_VERIFY_TOKEN.
  • Se preencher, as variáveis entram no YAML antes do deploy. Se não quiser, basta pressionar ENTER para pular.

Evolution API

Versão estável usada: v2.2.3 com CONFIG_SESSION_PHONE_VERSION=2.3000.1023015479 e CONFIG_SESSION_PHONE_CLIENT=CAPTIOAI.
